# AutoNova Imperium | Luxury Car Dealer System

**AutoNova Imperium** es una aplicación web moderna diseñada para la gestión y visualización de un catálogo de vehículos. El proyecto combina una interfaz estética en modo oscuro con funcionalidades interactivas de comercio electrónico.

---

## ✨ Características Principales

- 🖼️ **Hero Slider Dinámico:** Galería de imágenes en la cabecera que rotan automáticamente.
- 🔍 **Búsqueda y Filtros Pro:** Filtrado por categorías (SUV, Sedán, Deportivo) y búsqueda por marca o modelo en tiempo real.
- 🛒 **Carrito de Compras:** Gestión de selección de vehículos con persistencia de datos mediante `localStorage`.
- 💰 **Simulador de Financiación:** Cálculo automático de cuotas mensuales basado en una tasa de interés anual del 7%.
- 💱 **Multimoneda:** Conversión visual de precios a USD, EUR, GBP y JPY.
- 📄 **Generación de PDF:** Exportación de cotizaciones profesionales utilizando la librería `jsPDF`.
- ⚡ **Interfaz Premium:** Diseño responsivo con efectos de zoom dinámico y animaciones de entrada (Scroll Reveal).

---

## 🛠️ Tecnologías Utilizadas

* **HTML5 / CSS3:** Estructura y diseño personalizado con variables modernas.
* **Bootstrap 5:** Framework para el layout responsivo y componentes de UI (Modales, Navbar).
* **JavaScript (ES6+):** Lógica del negocio, manipulación del DOM y persistencia de datos.
* **jsPDF:** Librería externa para la creación de documentos PDF desde el cliente.
* **Font Awesome:** Iconografía de alta calidad.
